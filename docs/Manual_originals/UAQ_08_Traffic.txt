CarMaker Reference Manual Version 14.0.1<preQ>.Long.StepIdx Current longitudinal maneuver step Id, -11 for default mane uver steps
and -12 if the object is deactivated. Special treatment:
- if a start condition for the traffic object is specified and not yet true
(object is still deactivated), setting StepIdx to a value greater or equal to
0 bypasses the start condition and starts the maneuver control life cycle
- if the target maneuver step Id is greater than the number of steps of the
current active maneuver, the active maneuver is ended
- if the object after all maneuvers are completed is not deactivated/hid-
den, the maneuver control life cycle can be restarted with a value greater
or equal to 0
<preQ>.Lat.StepIdx Current longitudinal maneuver step Id, -11 for default man euver steps
and -12 if the object is deactivated. Special treatment:
- if a start condition for the traffic object is specified and not yet true
(object is still deactivated), setting the StepIdx to a value greater or equal
to 0 bypasses the start condition and starts the maneuver con trol life
cycle
- if the target maneuver step Id is greater than the number of steps of the
current active maneuver, the active maneuver is ended
- if the object is not deactivated/hidden after all maneuvers are complet-
ed, the maneuver control life cycle can be restarted with a value greater
or equal to 0
